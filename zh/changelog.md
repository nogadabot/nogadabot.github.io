# 更新日志

最新版本显示在最上方。每次打开应用都会自动更新到最新版本 — 安装请参阅 [下载](getting-started/download.md)。

## v0.9.29

- 所有受支持的铸造平台现在都能从发射到确认始终绑定你选择的钱包、链、价格、数量、接收地址与交易内容，铸造更加可靠。
- 高负载下发射更快：提前完成网络与 Gas 准备，减少大批钱包的不必要排队，并且首笔交易前不增加新的在线等待。
- 停止、重启与 Boost 现在只会继续或替换属于该任务的交易，避免重复铸造或误改其他待处理交易。
- NFT 转账、挂单、报价、资金分发与归集在超时后可正确恢复，并更准确地显示实际链上结果。
- 提升了定时执行、钱包管理、许可证、支付、更新、备份、状态界面以及 Telegram 机器人的整体稳定性。

## v0.9.28

- 铸造更快：移除首笔交易前的延迟（自动 Gas 准备），开售瞬间发射更迅速，力争抢到第一个区块。
- 按你设置的 Gas 一次性发射：应用不再每隔几秒自动提高 Gas，而是按你设置的 Gas（自动/自定义）发射一次；想提高请用 Boost 按钮。
- 铸造成功现在会在状态栏正确显示（绿色）——已上链的铸造不再被误标为失败。

## v0.9.27

- ERC-20 计价 Drop 支付更准确：Manifold 与 TLStacks 铸造现在会发送准确的原生平台费，ERC-20 计价的 Drop 不再回退。
- Manifold 白名单 Drop 现可正常铸造（使用白名单领取的链上费用）。
- 修复粘贴 OpenSea 链接时偶发的红色错误：应用现在会自动在你的 RPC 间重试，而不是首次就失败。
- 更安全的自定义 Gas：设置最高费用后，小费会保持在其范围内，避免网络拒绝铸造。

## v0.9.26

- 更可靠的铸造：在每条铸造路径上，价格、金额、Gas 与 ERC-20 处理都更准确。
- 更快的发射：预热 RPC 连接并精简了发射路径。
- 更稳健：改进重试、状态显示更清晰，Drop 数据变动时可自愈。
- 新增：白名单检查可使用运营者发布的外部检查器，根据粘贴的 slug 或链接自动路由。
- 全应用范围内的大量稳定性与准确性修复。

## v0.9.25

- 新增：Drop 页面 — 扫描热门链上正在开放或刚部署的 SeaDrop 铸造，点击即可直接载入 Smart Mint。
- 多钱包白名单与签名 Drop 铸造更快了：现在按 Drop 并行获取每个钱包的凭证并合并处理，钱包再多也无需逐个等待即可发射。
- 新增：同一 Drop 的各钱包任务在任务页折叠为一张卡片 — 查看 Drop、阶段、钱包数量与实时进度，并一键全部启动 / 停止。
- 新增：按任务调节 Gas — 为单个任务设置小费倍数与最低优先费，并为连发模式提供 15/30/60% 快捷加成预设。
- 修正了韩文与中文界面中若干仍显示为英文的标签。

## v0.9.24

- 新增：白名单（签名）Drop 可使用你自己的 OpenSea API 密钥。注册一次后，应用与 Telegram 机器人将通过 OpenSea 官方 Drops API（每账户每分钟 120 次）获取凭证，而非共享 swap。公开铸造仍无需密钥。
- 新增：可自定义代理测试 URL，用你喜欢的任意地址检查代理（默认 api.ipify.org）。
- 新增：在钱包页面一键将 WETH 解包换回 ETH。
- 新增：应用、用户指南与 Telegram 机器人现已全面支持简体中文 — 在设置中切换语言。
- 优化了英文、韩文与中文的措辞，使其更清晰自然。

## v0.9.23

- 新增：在钱包页面一键检查钱包是否存在 EIP-7702 委托（部分铸造会拒绝的智能账户升级）并撤销。
- 新增：在设置中登出。它会停止所有运行中的任务并用你的密钥锁定应用，但保留全部钱包、RPC 和设置。
- 新增：分发资金时可为每个钱包设置随机金额，避免给所有钱包发送完全相同的数额。
- 新增：可直接归集到外部地址（交易所充值地址或冷钱包），并新增 Gas 预估的 ETH / USD 一键切换。
- 白名单铸造在大规模下对频率限制更友好：现在会将凭证请求分散到你的多个代理，连发监控也更紧密地跟踪铸造开放时间。
- 优化了应用和应用内指南的措辞，使其更清晰自然。
