# 更新日志

最新版本显示在最上方。每次打开应用都会自动更新到最新版本 — 安装请参阅 [下载](getting-started/download.md)。

## v0.9.24

- 新增：白名单（签名）Drop 可使用你自己的 OpenSea API 密钥。注册一次后，应用与 Telegram 机器人将通过 OpenSea 官方 Drops API（每账户每分钟 120 次）获取凭证，而非共享 swap。公开铸造仍无需密钥。
- 新增：可自定义代理测试 URL，用你喜欢的任意地址检查代理（默认 api.ipify.org）。
- 新增：在钱包页面一键将 WETH 解包换回 ETH。
- 新增：应用、用户指南与 Telegram 机器人现已全面支持简体中文 — 在设置中切换语言。
- 优化了英文、韩文与中文的措辞，使其更清晰自然。

## v0.9.23

- 新增：在钱包页面一键检查钱包是否存在 EIP-7702 委托（部分铸造会拒绝的智能账户升级）并撤销。
- 新增：在设置中登出。它会停止所有运行中的任务并用你的密钥锁定应用，但保留全部钱包、RPC 和设置。
- 新增：分发资金时可为每个钱包设置随机金额，避免给所有钱包发送完全相同的数额。
- 新增：可直接归集到外部地址（交易所充值地址或冷钱包），并新增 Gas 预估的 ETH / USD 一键切换。
- 白名单铸造在大规模下对频率限制更友好：现在会将凭证请求分散到你的多个代理，连发监控也更紧密地跟踪铸造开放时间。
- 优化了应用和应用内指南的措辞，使其更清晰自然。
