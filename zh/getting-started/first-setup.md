# 3. 初始必要设置（5分钟）

完成激活后，请在铸造前按照以下顺序进行设置。只需完成 **①钱包 → ②RPC → ③ Gas 确认**，即可立即开始铸造，④为可选项。

## ① 添加钱包（必须）

添加用于铸造的钱包。前往 **钱包** 菜单：

* **创建钱包**：由应用自动生成新钱包（私钥也会自动保存）。
* **导入私钥**：粘贴已有钱包的私钥进行导入。

> 🔐 私钥**仅在您的PC上加密存储**。即便如此，**请使用小额的"燃烧钱包"，而非存有大量资金的主钱包**进行铸造。

详细说明 → [钱包 (Wallets)](../app-guide/wallets.md)

## ② 添加 RPC（必须）

RPC 是应用与区块链通信的"通道"。前往 **RPC** 菜单添加端点。

* **免费**：在 [Chainlist](https://chainlist.org) 上复制所需链的公共 RPC 地址并粘贴
* **付费（推荐）**：Alchemy、DRPC 等，速度快且稳定，无速率限制（铸造竞争越激烈，越发重要）

添加后，请通过 **测试** 功能确认 ping 值是否较低。

详细说明 → [RPC](../app-guide/rpc.md) · 推荐 → [RPC/节点链接](../resources/nodes.md)

## ③ 确认 Gas（必须）

前往 **工具** 菜单中的 **Gas 计算器**，提前确认当前 Gas 价格及单次铸造费用。钱包中需持有足够的 ETH 才能发出交易。

> 💡 设置较高的自定义 Gas 时，请安全起见在钱包中保留**比预计费用多出 20~50% 的 ETH**。复杂的合约可能消耗更多 Gas。

详细说明 → [工具（Gas 计算器）](../app-guide/tools.md) · [Gas 设置完全指南](../minting/gas.md)

## ④（可选）代理 · API 密钥

**大多数铸造情况下无需使用。** 仅在以下情况需要：

* **代理**：拥有大量钱包，或进行基于网站的操作（如 WL 网站）时用于分散 IP → [代理](../app-guide/proxies.md)
* **OpenSea / Alchemy / Etherscan / 验证码 API 密钥**：用于 NFT 持仓查询、盈亏统计、挂单等附加功能。**铸造本身无需 API 密钥。** → [设置](../app-guide/settings.md)
