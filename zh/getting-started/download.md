# 下载 & 安装

## 首次安装

请从以下链接获取最新安装文件。

> ### [⬇️ Nogada 安装文件下载 (Windows)](https://nogada-lic.duckdns.org/updates/nogada-app-setup.exe)

1. 从上方链接下载 `nogada-app-setup.exe`。
2. 运行文件进行安装。（若出现 Windows SmartScreen 警告，请点击**更多信息 → 仍要运行**，这是正常现象。）
3. 打开应用，输入购买时收到的**许可证密钥**进行激活。（参考 [许可证激活](license.md)）

安装只需进行一次。详细安装流程请参阅 [安装指南](installation.md)。

## 已安装的情况：自动更新

如果已安装 Nogada，**无需单独下载新版本。** 每次启动应用时，会自动检查最新版本并自行更新。如需了解更新内容，请查看 [更新日志](../changelog.md)。

## 系统要求

- Windows 10 / 11 (64-bit)
