# 4. 推荐附加设置（可选）

完成必要设置（钱包 · RPC · Gas）后，以下是**能提高铸造成功率、速度和便利性的"锦上添花"设置**。全部为可选项，但如果目标是竞争激烈的铸造，强烈推荐 ⭐ 标记的项目。

> 💡 不必一次全部完成。**竞争铸造请先从 ⭐ 3 项开始**，其余在需要时再添加。

## 🏆 竞争铸造推荐（强烈建议）

### ⭐ 付费专属 RPC
公共 RPC 容易触发速率限制。低供应量先到先得（FCFS）的铸造往往在 1\~2 个区块内结束，因此速度更快、无限制的**付费 RPC** 更具优势。
* 在 Alchemy · DRPC 等平台申请 → 添加到 **RPC** 菜单 → 通过**组测试**只保留延迟低的节点
* 详情 → [RPC](../app-guide/rpc.md) · [RPC / 节点链接](../resources/nodes.md)

### ⭐ 多 RPC 广播
将交易**同时**发送到多个 RPC，提高快速打包进区块的概率。
* 开启 **设置 → 引擎 → 多 RPC 广播**
* > ⚠️ 创建任务时，不要忘记**勾选该任务要使用的 RPC**。（未选择则使用公共 RPC 执行。）

### ⭐ 预设快速发射默认值
直播铸造等紧急情况下，每次临时选择钱包 · RPC 会浪费时间。
* 在 **设置 → 快速发射** 中预先指定默认钱包 · RPC → 一键发射
* 详情 → [任务 (Tasks)](../app-guide/tasks.md)

## ⚙️ 稳定 · 高级（按需配置）

### Flashbots（私有打包）
希望减少抢跑风险和失败交易时使用。
* **设置 → 引擎 → Flashbots**：开启打包 + 调整 window / priority

### Gas 默认策略
* **设置 → 引擎 → Gas**：通过自动小费倍数（×）和最低 priority（gwei 下限）预设日常 Gas 偏好
* 详情 → [Gas 设置完全指南](../minting/gas.md)

### 代理
钱包数量非常多，或进行白名单铸造操作时用于分散 IP。
* 在 **代理** 菜单中添加 → 详情 → [代理](../app-guide/proxies.md)

## 🔔 便利 · 通知

### API 密钥（附加功能）
铸造本身**不需要**，但开启后会更方便：

| 密钥 | 启用的功能 |
|---|---|
| **OpenSea** | 上架状态 · 最佳报价 · 上架/接受 |
| **Alchemy** | NFT 持仓列表 · 盈亏（PnL） |
| **Etherscan** | 导入 ABI · 区块浏览器查询 |
| **验证码**（CapMonster 等） | 自动解决验证码（仅限有需要的铸造） |

详情 → [设置 → 集成](../app-guide/settings.md)

### Discord Webhook 通知
铸造成功/失败时立即通过 Discord 接收通知 → **设置 → 集成 → Discord Webhook**

### 开机自动运行
PC 登录时让应用**在后台自动运行**，保持机器人 · 通知 · 自动备份始终在线 → **设置 → 通用 → 开机自动运行**

### Telegram 机器人（即使关闭应用也可 24 小时运行）
外出时也能通过手机进行铸造 · 钱包管理 → 详情 → [Telegram 机器人](../telegram/telegram-bot.md)

---

> ✅ 总结：**竞争铸造 = 先搞定 ⭐（付费 RPC · 多 RPC · 快速发射）**。通知/便利功能慢慢配置。全部都不是必须的。
