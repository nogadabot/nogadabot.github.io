# 2. 许可证激活

使用 Nogada 需要**许可证密钥**。获取密钥 → 输入 → 激活，三步即可完成。

## 1) 购买

在运营者的 **Whop 销售页面**完成订阅付款。

## 2) 获取许可证密钥

付款完成后，可通过以下**两种方式之一**获取密钥（两种都可以使用）：

### 📩 方式 A：邮件（自动，推荐）

付款后，**许可证密钥将自动发送至您购买时使用的邮箱**。（几分钟内送达：**请务必检查垃圾邮件文件夹**。）

密钥格式如下：`NOGADA-XXXX-XXXX-XXXX-XXXX`

### 🤖 方式 B：Telegram（未收到邮件时）

1. 在 Telegram 中搜索 **@NOGADA\_Mint\_Bot**，打开对话。
2. 按如下格式输入（使用购买时的邮箱）：

   ```
   /redeem 您的邮箱@example.com
   ```
3. 机器人将查找并发送您的密钥。

> 💡 **显示"未找到密钥"时**：请确认购买时使用的邮箱与输入的邮箱完全一致。仍然无效请联系运营者。

## 3) 在应用中输入密钥 → 激活

1. 打开 Nogada 应用。
2. 在首页（或**设置 → 许可证**）的输入框中粘贴密钥。
3. 点击**激活**按钮。

![设置 - 许可证选项卡](../images/settings-license.png)

> *设置 → 许可证。激活后会显示**设备绑定**和**停用**按钮（迁移到其他电脑前请先点击）。首次运行时，此处显示的是**密钥输入框 + 激活**按钮。*

激活后，应用的所有功能将解锁。右下角显示**认证 ●**（绿点）即为正常。

## ⚠️ 重要：仅限单台设备

* **一个许可证密钥只能在一台电脑上使用**。
* **需要更换电脑时**：先在原电脑的应用中**停用（解绑设备）**许可证，再在新电脑上重新激活即可。（若原电脑无法启动应用，请联系运营者申请重置设备。）

## 退款政策

退款仅适用于**无法安装**的情况。铸造成功与否受 Gas、剩余数量、项目状况等诸多因素影响，**无法保证，不在退款范围内。**

> ✅ **总结**：Whop 付款 → 通过邮件（或 Telegram /redeem）获取密钥 → 粘贴至应用并激活 → 确认认证 ●。
