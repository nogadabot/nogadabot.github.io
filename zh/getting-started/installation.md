# 1. 安装

Nogada 是一款适用于 Windows 10/11 的桌面应用程序。安装只需 2~3 分钟即可完成。

## 1) 获取安装文件

请从运营者提供的**下载链接**获取安装文件。

* `Nogada-Setup-x.x.x.exe`，**安装程序**（推荐）。双击后将自动安装。

> 💡 **下载被拦截或无法下载时**：浏览器或杀毒软件（Windows Defender）可能会以"来源不明的程序"为由进行拦截。这在未经签名的新应用中很常见。请参考下方"跳过警告"的说明。

## 2) 跳过安装过程中的警告（这是正常现象）

首次运行时，可能会出现蓝色的 **"Windows 已保护你的电脑"（SmartScreen）** 警告。

1. 点击**更多信息**。
2. 点击下方出现的**仍要运行**按钮。

> 💡 如果杀毒软件删除或拦截了文件：请在杀毒软件设置中将 Nogada 添加到**例外（允许）列表**，或在安装期间暂时关闭杀毒软件。安装完成后可以重新开启。

## 3) 进行安装

按照安装向导的提示操作（下一步 → 下一步 → 安装）即可。完成后，桌面/开始菜单中会出现 **Nogada** 图标。

## 4) 首次运行

启动 Nogada 后，会首先显示**许可证密钥输入界面**。如果还没有密钥，请前往 → [2. 许可证激活](license.md)。

## 5) 自动更新

安装一次后，**之后无需再操心。** 有新版本发布时，应用启动时会自动下载并更新。（当前版本显示在应用左下角。）

> ✅ **总结**：获取安装文件 → 跳过警告（更多信息 → 仍要运行）→ 安装 → 输入许可证 → 完成。之后更新自动进行。
