# 公开铸造

这是任何人都可以参与的公开铸造。在 Nogada 中有两种操作方式。

> 💡 竞争激烈的公开铸造，核心在于 **连发（连续发送）+ 快速 Gas + 定时启动** → [铸造方式完全指南](modes.md)

## 方式 A：启动台/市场链接（最简单）

如果是 OpenSea、Transient 等启动台，**只需粘贴链接**即可。

1. **任务** → `+ 新建任务`。
2. 在目标标签中选择 **OpenSea 活动** 或 **启动台**。
3. 在**合约地址栏中粘贴铸造链接**。
4. 应用将**自动识别铸造阶段（phase）** → 选择对应阶段。
5. 填写钱包、RPC、Gas，点击**创建** → **执行**。

> 💡 如果铸造已经实时开始，自动识别是最快、最方便的方式。

## 方式 B：直接填写合约（手动）

对于链接无法自动识别的普通合约，需要手动指定函数。

1. 准备要铸造项目的**合约地址**（从 Etherscan 等处获取）。
2. **任务** → `+ 新建任务` → 目标选择**合约**。
3. 输入合约地址，点击 **导入 ABI**，从函数列表中选择 `mint` / `publicMint` 等。
4. 填写**参数**。公开铸造通常只有一个 **数量（quantity）**。
5. 在**金额（ETH）**中填入铸造价格（免费则填 0）。
6. 设置钱包、RPC、Gas → **创建** → **执行**。

### 直接输入 hex（高级）

也可以从他人的铸造交易中复制**输入数据（hex）**来使用。

* 在 Etherscan 上找到该合集的铸造 tx → **Input Data** → "View Input As → Original" 复制 hex
* 在任务编辑器中勾选 **Raw calldata** 并粘贴该 hex
* 只需保证合约地址 + 总金额（铸造价 × 数量）正确即可

> ⚠️ **包含 receiver 地址或 token id 的函数**请谨慎操作。复制错误可能导致铸造到他人钱包。只有简单的 `mint(数量)` 类函数才可以直接复制使用。

> 💡 如果对函数/hex 感到困惑，可以向 ChatGPT·Claude 提问"这个合约的铸造函数和参数是什么？"，会得到详细解答。
