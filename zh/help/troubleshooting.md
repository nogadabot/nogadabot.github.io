# 故障排查

## 无法下载 / 文件消失了

* 杀毒软件（Windows Defender）或浏览器拦截是最常见的原因。请尝试**以 zip 格式下载**，或在杀毒软件中临时设置允许/例外后重新下载。
* 如果下载链接被拦截，请复制链接后**粘贴到新标签页**中打开。

## 安装文件无法运行

* 出现蓝色 SmartScreen 警告 → 点击**更多信息 → 仍要运行**。
* 仍然被拦截：右键单击安装文件 → **属性 → 勾选"取消锁定（Unblock）"→ 应用**。
* 在杀毒程序的**例外/信任列表**中添加 Nogada。

## 无法激活许可证

* 请确认密钥**不含空格、完整粘贴**（格式：`NOGADA-XXXX-XXXX-XXXX-XXXX`）。
* 请检查网络连接（激活需要连接服务器进行验证）。
* 提示"已在其他设备使用"→ 请在该设备上停用许可证，或联系运营者申请重置设备。

## 没有收到许可证密钥

* 请检查**邮件垃圾箱**。
* 在 Telegram **@NOGADA\_Mint\_Bot** 中输入 `/redeem 购买邮箱`。
* 提示"找不到密钥"时，请确认购买时使用的邮箱与输入的邮箱是否一致。仍未解决请联系运营者。

## 铸造交易未发出

* 确认**钱包余额**充足（铸造价格 + Gas，建议预留 20～50% 余量）。
* 确认 **RPC** 已连接且在任务中勾选。可在 [RPC] 界面进行测试。
* 确认 **Gas** 费用设置不过低。参阅 [Gas 指南](../minting/gas.md)。
* **Gas Limit 请留空**：手动填入过低数值会导致"out of gas"失败。

## 交易一直处于待处理（pending）状态

* 请使用**加速（🚀）**按钮以更高的 Gas 重新提交。→ [加速](../minting/boost.md)

## 无法导入 ABI

* 在[设置 → 集成](../app-guide/settings.md)中填入 **Etherscan API 密钥**可大幅提升成功率（未填写时会尝试通过 Sourcify 获取）。
* 未经验证的合约可能没有 ABI → 请直接手动填写函数签名或 hex。

## 铸造已开始但任务没有响应

* 启动后 5～10 秒内日志若未变为"发送等待中"，请**前往项目官网手动铸造**作为备用方案。项目方有时会在最后一刻更改配置。

> 若以上方法均无效，请**附上截图联系运营者**。根据 Windows 版本和杀毒软件类型，可能需要进行额外设置。
