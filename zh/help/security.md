# 安全与安全守则

以下是安全使用 NFT 机器人的核心原则。**请务必阅读。**

## 1. 私钥仅保存在您的电脑上

* 导入钱包的私钥以 **Windows DPAPI 加密，仅存储在本机**。
* 我们的服务器既不接收也不存储私钥。
* **导出密钥**是唯一可在屏幕上显示私钥的功能，请注意不要在周围有人、录屏或直播时暴露。

## 2. 始终使用小额钱包（Burner 钱包）铸造

* 大额资产请保存在**从未连接过任何 dApp 或应用的金库（Vault）钱包**中。
* 铸造、活动、空投请使用 **Burner 钱包**（即使丢失也无所谓的小额钱包）。仅此一点即可大幅降低风险。

## 3. Telegram 机器人钱包也应使用 Burner

* Telegram **机器人钱包保存在服务器上**，是用于 24 小时运行的 Burner 钱包。**请勿存入大额资金。**

## 4. 请勿泄露配对码或密钥

* Telegram **配对码**等同于密码。录屏或直播时绝对不能暴露，看到该码的人可以连接机器人并盗取机器人钱包内的资金。
* **授权密钥**也请勿共享（虽然仅限单台设备使用，他人无法使用，但仍请避免泄露）。

## 5. 警惕钓鱼：仅使用官方链接

* 使用机器人铸造可降低连接诈骗/钓鱼网站的风险，但**私信及陌生链接请始终保持警惕**。
* 下载与购买请仅通过**运营方提供的官方链接**进行。

## 6. 良好习惯总结

| 应做 ✅ | 不应做 ❌ |
|---|---|
| 使用 Burner 钱包铸造 | 使用主钱包/金库钱包铸造 |
| 大额资产保存在未连接的金库中 | 在机器人钱包或 Burner 钱包中存放大额资金 |
| 仅通过官方链接下载 | 点击私信或陌生链接 |
| 保密配对码与密钥 | 直播时暴露配对码或密钥 |

> web3 中没有百分之百的安全。养成这些习惯是保护您资产最可靠的方式。
