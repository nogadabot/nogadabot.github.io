# 常见问题 (FAQ)

## 支持哪些链？

支持以太坊主网（L1）以及主流 L2/EVM 链，包括 **Arbitrum、Optimism、Base、Polygon、BNB Chain、Avalanche、Linea、Abstract、Zora、Blast、Scroll** 等。列表中没有的 EVM 链也可以自定义添加。

## 我的钱包 100% 安全吗？

钱包安全是首要设计原则。导入钱包的**私钥经过加密（Windows DPAPI）后仅存储在您的本地电脑**，**不会上传至我们的服务器。** 只有您电脑上的 Nogada 才能解密私钥并对铸造交易进行签名。

不过，Web3 领域不存在绝对的 100% 安全。最佳实践是：**大额资产放入从不连接任何 dApp 的冷藏钱包**，铸造时使用**临时（小额）钱包**。→ [安全须知](security.md)

## 需要自己准备 RPC 节点吗？

是的，需要 RPC。**免费**方案可使用 [Chainlist](https://chainlist.org) 上的公共 RPC；竞争激烈的铸造建议使用**付费专属 RPC**（Alchemy/DRPC 等）。尤其是其他 EVM 链（Arb/Base 等），需要填入自己的 RPC 地址。→ [RPC 链接](../resources/nodes.md)

## 新手用起来难吗？

不难。按照[快速开始](../getting-started/installation.md)的 3 个步骤和[铸造指南](../minting/public.md)操作，点击几下即可完成铸造。不过成功率受 Gas、竞争激烈程度、项目状况等多种因素影响。

## 如何购买许可证？

在运营者的 **Whop 页面**完成订阅付款。→ [许可证激活](../getting-started/license.md)

## 付款后在哪里获取许可证密钥？

系统会**自动发送到购买时使用的邮箱**（请同时检查垃圾邮件文件夹）。如未收到，可在 Telegram **@NOGADA\_Mint\_Bot** 中输入 `/redeem 您的邮箱` 来领取。

## 可以在多台设备上使用吗？

不可以，**1 个许可证 = 1 台电脑**。若需更换设备，请先在旧电脑上停用许可证（解绑设备），再在新电脑上激活。

## 可以退款吗？

仅限**软件本身无法安装**的问题可申请协助。铸造成功与否受 Gas、剩余供应量等诸多变量影响，无法保证结果，因此不在退款范围内。

## 一定需要验证码/代理吗？

不需要。**铸造本身无需验证码、代理或 API 密钥。** 验证码仅在部分设有防护机制的铸造活动中需要，代理仅在使用多个账号操作网站时才有必要。

## 能保证铸造成功吗？

无法保证。低供应量的先到先得活动，成功与否取决于所有参与者的 Gas、RPC 速度、网络状况以及项目 API 状态，差距往往在毫秒之间。Gas 和 RPC 准备得越充分，成功概率越高。
