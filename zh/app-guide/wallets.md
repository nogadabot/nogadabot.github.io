# 钱包 (Wallets)

这是管理用于铸造的钱包的界面。您可以创建或导入钱包、查看余额以及转移资金。

![wallets](../images/wallets.png)

![钱包按钮放大](../images/wallets-detail.png)

> 🔍 *放大显示：**添加钱包**（创建·导入）与**创建机器人钱包**（24小时运行）。*

## 顶部

* **选择链**：选择查看哪条链的余额（Ethereum、Base 等）。
* **查看 ERC-20 余额**：输入特定代币地址后，该代币余额将显示在列中。
* **添加钱包**：添加新钱包。

## 🤖 机器人钱包（服务器）：用于 Telegram

界面上方的**机器人钱包（服务器）**部分是专供 Telegram 使用的钱包，**即使关闭应用也能24小时运行**。

* **创建机器人钱包**：创建用于 Telegram 铸造的钱包。
* **隐藏 / 刷新**：切换地址显示隐藏 / 刷新余额。

> 🔐 机器人钱包是**保存在服务器上的小额临时（Burner）钱包**。它与应用内的普通钱包（私钥仅保存在您的电脑上）相互独立。**请勿存入大额资金。** → [Telegram 机器人](../telegram/telegram-bot.md)

## 分组 & 钱包列表

* **分组栏（左侧）**：将钱包归入分组。`+ 新建分组`。
* **钱包表格**：复选框 · 名称 · 地址 · ETH · WETH。勾选后可批量操作多个钱包。
* **按名称/地址筛选**：钱包较多时可搜索。

## 按钮说明

| 按钮 | 功能 |
|---|---|
| **生成密钥（Generate）** | 自动生成 N 个新钱包（自动保存私钥） |
| **导入密钥（Import）** | **粘贴现有钱包的私钥**进行导入（多行 = 多个） |
| **资金管理（Manage Funds）** | 在钱包之间转移资金（详见下方说明） |
| **白名单查询** | 确认所选钱包是否在特定 Drop 的白名单中 |
| **刷新余额** | 重新加载余额 |

![导入密钥弹窗](../images/wallet-import.png)

> ***添加钱包 → 粘贴密钥**弹窗：每行粘贴一个私钥，然后点击**导入**。*

> 🔐 **私钥仅以加密方式保存在您的电脑上**（不会传输至服务器）。即便如此，铸造时仍建议使用**临时（Burner）钱包**。

## 💸 资金管理（Disperse / Consolidate）

用于将 Gas 分发到多个钱包，或将分散的余额归集到一个钱包。

* **分发（Disperse）**：从一个钱包向多个钱包发送 ETH（铸造前充值 Gas）
* **归集（Consolidate）**：从多个钱包将余额汇入一个钱包（使用 `send-max` 回收几乎全部余额）

### 🎯 实战示例：向钱包批量发送 Gas

铸造前，将**1个有资金的钱包 → 所有铸造钱包**的 Gas 发送出去：

![分发示例](../images/wallet-fund-example.png)

| # | 步骤（示例） |
|---|---|
| ① | 选择**分发（Disperse）**（一个 → 多个）。*（归集则相反：多个 → 一个）* |
| ② | **发送方**：存有资金的**那个钱包**（出发地） |
| ③ | **接收方**：需要接收 Gas 的钱包（目的地） |
| ④ | **各钱包金额**：每个钱包发送的 ETH 数量（例：`0.01`） |
| ⑤ | **开始分发**：执行转账。完成后显示成功/等待/失败状态，仅失败的可**重试**。 |

## 底部栏（选中钱包时）

对所选钱包可执行**复制地址 / 导出密钥（需确认）/ 删除（需确认）/ 清空**等操作。

> ⚠️ **导出密钥**是将私钥显示在屏幕上的唯一功能。请注意周围环境，**录屏或直播时切勿点击**。
