# NFT（持仓 · 市场）

查看已铸造的 NFT，并在 OpenSea 上进行买卖的界面。顶部有 **持仓 / 市场** 两个标签页。

![nft](../images/nft.png)

## 通用

* **选择链 / 选择分组 / 刷新**：选择要查看的链和钱包分组，并刷新 NFT 列表。

## 📦 持仓（Holdings）标签页

显示我的各个钱包所持有的 NFT 列表。

> ⚙️ 加载持仓列表需要 **Alchemy URL**。请在[设置 → Setup](../app-guide/settings.md)中按链填写。（未填写时会显示"Alchemy URL not configured"。）

## 🏷️ 市场（Market）标签页

处理 OpenSea 上架单与报价。（大部分功能需要 **OpenSea API 密钥**才能使用。→ [设置](../app-guide/settings.md)）

* **实时上架单**：将我已加载的 NFT 与 OpenSea 上的开放订单进行对比显示。点击"重新检查"可刷新。
* **分组批量取消（Cancel All）**：一次性取消该分组内所有钱包的开放上架单（每个钱包发送 1 笔交易）。每个分组旁会显示钱包数量。
* **藏品最高报价**：预览各藏品当前的最高报价。
* （创建上架单 / 接受报价等操作也在此处进行。）

> 💡 **如果只是用来铸造，不了解此界面也没关系。** 这是一个高级界面，供您将铸造的 NFT 直接在 OpenSea 上出售或管理。
