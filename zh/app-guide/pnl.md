# 盈亏 (PnL)

计算铸造所花费的成本与获得的收益，并生成**便于分享的卡片图片**的页面。

![pnl](../images/pnl.png)

## 使用方法

1. **集合**：输入要计算盈亏的集合所在的链 + 合约地址。
2. **钱包**：勾选要纳入计算的钱包（组）。
3. **（可选）卡片背景**：使用默认值，或在 3 个图片槽中上传所需背景（PNG/JPG/WebP，最大 10MB）。槽位仅保存在本地。
4. **计算**：结果将生成为**盈亏卡片**。（已实现盈亏 / ROI / Gas / 总支出 / 变现金额）

> ⚙️ 盈亏计算同样需要 **Alchemy URL**。请在[设置 → Setup](../app-guide/settings.md)中按链分别填写。

> 💡 绿色 = 盈利，红色 = 亏损。生成的卡片可直接截图分享。
