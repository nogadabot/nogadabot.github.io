# 代理 (Proxies)

代理可以隐藏您的真实 IP，防止请求从同一 IP 集中发出。**大多数铸造场景下无需使用**，但在钱包数量较多或需要进行网页端操作时非常有用。

![proxies](../images/proxies.png)

## 什么时候需要？

* 使用数十乃至数百个钱包/账号进行**网页端操作（白名单网站、Twitter、Discord）**时 → 若所有请求来自同一 IP，可能触发机器人检测、封锁或封号，使用代理可将流量分散。
* 如果仅是直接向合约发起铸造，通常**不需要**代理。

## 格式

代理通常使用以下格式：

```
host:port:username:password
```

可以逐行粘贴多个代理。

## 界面组成 / 按钮

* **分组栏（左侧）**：管理代理分组。`+ 新建分组`。
* **导入代理**：按上述格式粘贴后添加。
* **全部测试**：检查代理是否可用及响应速度。
* **全部删除**：清空当前分组。
* **按主机筛选**：代理较多时可快速搜索。

## 代理类型（简介）

* **住宅代理（Residential）**：使用真实家庭 IP。按 GB 计费，剩余流量不过期。速度较慢，适合新手入门。
* **ISP 代理**：速度更快、更稳定，通常按 IP 按月计费。
