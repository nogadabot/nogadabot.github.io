# 4. 추천 부가 세팅 (선택)

필수 세팅(지갑 · RPC · 가스)을 끝냈다면, 아래는 **민팅 성공률 · 속도 · 편의를 높여주는 "있으면 좋은" 세팅**입니다. 전부 선택이지만, 경쟁이 치열한 민팅을 노린다면 ⭐ 표시는 강력 추천합니다.

> 💡 한 번에 다 안 해도 됩니다. **경쟁 민팅이면 ⭐ 3개부터**, 나머지는 필요할 때 추가하세요.

## 🏆 경쟁 민팅이라면 (강력 추천)

### ⭐ 유료 전용 RPC
공개 RPC는 속도 제한에 잘 걸립니다. 저공급 선착순(FCFS)은 1~2블록 만에 끝나서, 빠르고 제한 없는 **유료 RPC**가 유리합니다.
* Alchemy · DRPC 등에서 발급 → **RPC** 메뉴에 추가 → **그룹 테스트**로 핑 낮은 것만 남기기
* 자세히 → [RPC](../app-guide/rpc.md) · [RPC / 노드 링크](../resources/nodes.md)

### ⭐ 멀티-RPC 브로드캐스트
트랜잭션을 여러 RPC로 **동시에** 쏴서 블록에 빨리 들어갈 확률을 높입니다.
* **설정 → 엔진 → 멀티-RPC 브로드캐스트** 켜기
* > ⚠️ 작업을 만들 때 **그 작업에 쓸 RPC들을 체크**하는 걸 잊지 마세요. (안 고르면 공개 RPC로 실행됩니다.)

### ⭐ 퀵 파이어 기본값 미리 지정
라이브 민트처럼 급할 때 매번 지갑 · RPC를 고르면 늦습니다.
* **설정 → 퀵 파이어**에서 기본 지갑 · RPC를 미리 지정 → 한 번에 발사
* 자세히 → [작업 (Tasks)](../app-guide/tasks.md)

## ⚙️ 안정 · 고급 (상황별)

### Flashbots (프라이빗 번들)
프론트러닝 노출과 실패 트랜잭션을 줄이고 싶을 때.
* **설정 → 엔진 → Flashbots**: 번들 켜기 + window / priority 조정

### 가스 기본 정책
* **설정 → 엔진 → 가스**: 자동 팁 배수(×)와 최소 priority(gwei 하한)로 평소 가스 성향을 정해둠
* 자세히 → [가스 설정 완전정복](../minting/gas.md)

### 프록시
지갑이 아주 많거나 WL 사이트 작업을 할 때 IP 분산용.
* **프록시** 메뉴에 추가 → 자세히 → [프록시](../app-guide/proxies.md)

## 🔔 편의 · 알림

### API 키 (부가 기능)
민팅 자체엔 **필요 없지만** 켜두면 편한 것들:

| 키 | 켜주는 기능 |
|---|---|
| **OpenSea** | 리스팅 상태 · 베스트 오퍼 · 리스팅/수락 |
| **Alchemy** | NFT 보유목록 · 손익(PnL) |
| **Etherscan** | ABI 가져오기 · 익스플로러 조회 |
| **캡차**(CapMonster 등) | 캡차 자동 풀기 (필요한 민팅에만) |

자세히 → [설정 → 연동](../app-guide/settings.md)

### Discord 웹훅 알림
민팅 성공/실패를 디스코드로 즉시 받기 → **설정 → 연동 → Discord 웹훅**

### 시작 시 자동 실행
PC 로그인 시 앱을 **백그라운드로 자동 실행**해 봇 · 알림 · 자동 백업이 항상 돌게 → **설정 → 일반 → 시작 시 자동 실행**

### 텔레그램 봇 (앱 꺼도 24시간)
외출 중에도 폰으로 민팅 · 지갑 관리 → 자세히 → [텔레그램 봇](../telegram/telegram-bot.md)

---

> ✅ 정리: **경쟁 민팅 = ⭐(유료 RPC · 멀티-RPC · 퀵파이어)** 먼저. 알림/편의는 천천히. 다 필수는 아닙니다.
