# 다운로드 & 설치

## 처음 설치하는 경우

아래 링크에서 최신 설치 파일을 받으세요.

> ### [⬇️ Nogada 설치 파일 다운로드 (Windows)](https://nogada-lic.duckdns.org/updates/nogada-app-setup.exe)

1. 위 링크에서 `nogada-app-setup.exe`를 다운로드합니다.
2. 파일을 실행해 설치합니다. (Windows SmartScreen 경고가 뜨면 **추가 정보 → 실행**을 누르세요. 정상입니다.)
3. 앱을 열고, 구매 시 받은 **라이선스 키**를 입력해 활성화합니다. ([라이선스 활성화](license.md) 참고)

설치는 한 번만 하면 됩니다. 자세한 설치 과정은 [설치하기](installation.md)에서 볼 수 있습니다.

## 이미 설치한 경우: 자동 업데이트

이미 Nogada가 설치돼 있다면 **새 버전을 따로 받을 필요가 없습니다.** 앱을 켤 때마다 최신 버전을 자동으로 확인하고 업데이트합니다. 무엇이 바뀌었는지는 [변경사항](../changelog.md)에서 확인하세요.

## 시스템 요구사항

- Windows 10 / 11 (64-bit)
