# 1. 설치하기

Nogada는 Windows 10/11용 데스크톱 앱입니다. 설치는 2~3분이면 끝납니다.

## 1) 설치 파일 받기

운영자가 안내한 **다운로드 링크**에서 설치 파일을 받으세요.

* `Nogada-Setup-x.x.x.exe` — **설치 프로그램**(추천). 더블클릭하면 자동으로 설치됩니다.

> 💡 **다운로드가 막히거나 안 될 때**: 브라우저나 백신(Windows Defender)이 "출처를 알 수 없는 프로그램"이라며 막을 수 있습니다. 이는 서명되지 않은 신규 앱에서 흔한 일입니다. 아래 "경고 넘기기"를 참고하세요.

## 2) 설치 중 경고 넘기기 (정상입니다)

처음 실행하면 파란색 **"Windows의 PC 보호"(SmartScreen)** 경고가 뜰 수 있습니다.

1. **추가 정보**를 클릭하세요.
2. 아래에 나타나는 **실행** 버튼을 누르세요.

> 💡 백신이 파일을 지우거나 막으면: 백신 설정에서 Nogada를 **예외(허용) 목록**에 추가하거나, 설치 중 잠시 백신을 꺼두세요. 설치가 끝나면 다시 켜도 됩니다.

## 3) 설치 진행

설치 마법사의 안내(다음 → 다음 → 설치)를 따르면 됩니다. 끝나면 바탕화면/시작 메뉴에 **Nogada** 아이콘이 생깁니다.

## 4) 첫 실행

Nogada를 실행하면 **라이선스 키 입력 화면**이 먼저 나타납니다. 키가 아직 없다면 → [2. 라이선스 활성화](license.md)로 이동하세요.

## 5) 자동 업데이트

한 번 설치하면 **다음부터는 신경 쓸 게 없습니다.** 새 버전이 나오면 앱이 켜질 때 자동으로 받아서 갱신합니다. (앱 하단 왼쪽에 현재 버전이 표시됩니다.)

> ✅ **정리**: 설치 파일 받기 → 경고 넘기기(추가 정보 → 실행) → 설치 → 라이선스 입력 → 끝. 이후 업데이트는 자동.
