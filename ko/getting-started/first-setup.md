# 3. 초기 필수 세팅 (5분)

활성화를 마쳤다면, 민팅 전에 딱 이 순서로 세팅하세요. **①지갑 → ②RPC → ③가스 확인**만 하면 바로 민팅할 수 있고, ④는 선택입니다.

## ① 지갑 추가 (필수)

민팅할 지갑을 넣습니다. **지갑** 메뉴로 가서:

* **지갑 생성** — 앱이 새 지갑을 만들어 줍니다 (개인키도 자동 보관).
* **키 가져오기** — 이미 쓰던 지갑의 개인키를 붙여넣어 가져옵니다.

> 🔐 개인키는 **당신 PC에만 암호화 저장**됩니다. 그래도 **민팅엔 큰돈이 든 메인 지갑 말고 버너(소액) 지갑**을 쓰세요.

자세히 → [지갑 (Wallets)](../app-guide/wallets.md)

## ② RPC 추가 (필수)

RPC는 앱이 블록체인과 대화하는 "통로"입니다. **RPC** 메뉴로 가서 엔드포인트를 추가하세요.

* **무료**: [Chainlist](https://chainlist.org)에서 원하는 체인의 공개 RPC 주소를 복사해 넣기
* **유료(추천)**: Alchemy, DRPC 등 — 빠르고 안정적, 속도 제한 없음 (경쟁이 치열한 민팅일수록 중요)

추가한 뒤 **테스트**로 핑(ping)이 낮은지 확인하세요.

자세히 → [RPC](../app-guide/rpc.md) · 추천 → [RPC/노드 링크](../resources/nodes.md)

## ③ 가스 확인 (필수)

**도구** 메뉴의 **가스 계산기**로 지금 가스 시세와 1회 민팅 비용을 미리 확인하세요. 지갑에 넉넉한 ETH가 있어야 트랜잭션이 나갑니다.

> 💡 커스텀 가스를 높게 설정할 땐, 안전하게 **예상 비용보다 20~50% 많은 ETH**를 지갑에 두세요. 복잡한 컨트랙트는 가스를 더 먹을 수 있습니다.

자세히 → [도구 (가스 계산기)](../app-guide/tools.md) · [가스 설정 완전정복](../minting/gas.md)

## ④ (선택) 프록시 · API 키

대부분의 민팅엔 **필요 없습니다.** 다음 경우에만:

* **프록시** — 지갑이 아주 많거나 웹사이트 기반 작업(WL 사이트 등)을 할 때 IP 분산용 → [프록시](../app-guide/proxies.md)
* **OpenSea / Alchemy / Etherscan / 캡차 API 키** — NFT 보유목록·손익·리스팅 등 부가 기능용. **민팅 자체는 키 없이 됩니다.** → [설정](../app-guide/settings.md)
