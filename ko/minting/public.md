# 퍼블릭 민트

누구나 참여하는 공개 민팅입니다. Nogada에서 두 가지 방법으로 할 수 있습니다.

> 💡 경쟁이 치열한 퍼블릭은 **연사(스팸) + 빠른 가스 + 예약 시작**이 핵심입니다 → [민트 방식 완전정복](modes.md)

## 방법 A: 런치패드/마켓 링크 (가장 쉬움)

OpenSea·Transient 같은 런치패드라면 **링크만 넣으면** 됩니다.

1. **작업** → `+ 새 작업`.
2. 대상 탭에서 **OpenSea 드롭** 또는 **런치패드** 선택.
3. **컨트랙트 주소 칸에 민팅 링크**를 붙여넣습니다.
4. 앱이 **민팅 단계(phase)를 자동 인식**합니다 → 단계를 고릅니다.
5. 지갑·RPC·가스 채우고 **생성** → **실행**.

> 💡 라이브로 이미 시작된 민트라면, 자동 인식이 가장 빠르고 편합니다.

## 방법 B: 컨트랙트 직접 (수동)

링크 자동인식이 안 되는 일반 컨트랙트는 직접 함수를 지정합니다.

1. 민팅할 프로젝트의 **컨트랙트 주소**를 준비합니다 (Etherscan 등에서).
2. **작업** → `+ 새 작업` → 대상 **컨트랙트**.
3. 컨트랙트 주소를 넣고 **ABI 가져오기**를 눌러 함수 목록에서 `mint` / `publicMint` 등을 고릅니다.
4. **인자**를 채웁니다. 퍼블릭 민트는 보통 **수량(quantity)** 하나입니다.
5. **금액(ETH)**에 민팅 가격을 넣습니다 (무료면 0).
6. 지갑·RPC·가스 설정 → **생성** → **실행**.

### hex로 직접 넣기 (고급)

다른 사람의 민팅 트랜잭션에서 **입력 데이터(hex)**를 복사해 쓸 수도 있습니다.

* Etherscan에서 그 컬렉션의 민팅 tx → **Input Data** → "View Input As → Original"로 hex 복사
* 작업 편집기에서 **Raw calldata**를 체크하고 그 hex를 붙여넣기
* 컨트랙트 주소 + 총 금액(민팅가 × 수량)만 맞추면 됩니다

> ⚠️ **receiver 주소나 token id가 들어가는 함수**는 조심하세요. 잘못 복사하면 남의 지갑으로 민팅될 수 있습니다. 단순 `mint(수량)` 류는 그대로 복사해도 안전합니다.

> 💡 함수/hex가 헷갈리면 ChatGPT·Claude에 "이 컨트랙트의 민팅 함수와 인자가 뭐야?"라고 물어보면 잘 알려줍니다.
