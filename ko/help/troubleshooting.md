# 문제 해결

## 다운로드가 안 돼요 / 파일이 사라져요

* 백신(Windows Defender)이나 브라우저가 막는 경우가 많습니다. **zip 형태로 받기**를 시도하거나, 백신에서 잠시 허용/예외 처리 후 다시 받으세요.
* 다운로드 링크가 막히면, 링크를 복사해 **새 탭에 붙여넣어** 열어보세요.

## 설치 파일이 실행이 안 돼요

* 파란 SmartScreen 경고 → **추가 정보 → 실행**.
* 그래도 막히면: 설치 파일 **우클릭 → 속성 → "차단 해제(Unblock)" 체크 → 적용**.
* 백신 프로그램의 **예외/안전 목록**에 Nogada를 추가하세요.

## 라이선스 활성화가 안 돼요

* 키를 **공백 없이 정확히** 붙여넣었는지 확인 (`NOGADA-XXXX-XXXX-XXXX-XXXX`).
* 인터넷 연결을 확인하세요 (활성화는 서버 인증이 필요).
* "다른 기기에서 사용 중"이라면 → 그 기기에서 비활성화하거나 운영자에게 기기 초기화 요청.

## 라이선스 키를 못 받았어요

* **이메일 스팸함**을 확인하세요.
* 텔레그램 **@NOGADA\_Mint\_Bot**에서 `/redeem 구매이메일` 입력.
* "키를 못 찾음"이면 구매 이메일과 입력 이메일이 같은지 확인. 그래도 안 되면 운영자 문의.

## 민팅 트랜잭션이 안 나가요

* **지갑 잔액**이 충분한지 (민팅가 + 가스, 여유 20~50%).
* **RPC**가 연결돼 있고 작업에 체크됐는지. [RPC] 화면에서 테스트.
* **가스**가 너무 낮지 않은지. [가스 가이드](../minting/gas.md).
* **가스 한도(Gas Limit)는 비워두기**: 직접 낮게 넣으면 "out of gas"로 실패합니다.

## 트랜잭션이 계속 대기(pending)예요

* **부스트(🚀)** 버튼으로 더 높은 가스로 다시 쏘세요. → [부스트](../minting/boost.md)

## ABI 가져오기가 안 돼요

* **Etherscan API 키**를 [설정 → 연동](../app-guide/settings.md)에 넣으면 더 잘 됩니다 (없으면 Sourcify로 시도).
* 검증 안 된 컨트랙트는 ABI가 없을 수 있습니다 → 함수/hex를 직접 넣으세요.

## 민팅이 시작됐는데 작업이 반응이 없어요

* 시작 후 5~10초 안에 로그가 "전송 대기"로 안 바뀌면, **프로젝트 사이트에서 수동 민팅**하세요(백업). 프로젝트가 막판 설정을 바꾸는 경우가 있습니다.

> 위 방법으로 안 되면, **스크린샷과 함께 운영자에게 문의**하세요. Windows 버전·백신 종류에 따라 추가 설정이 필요할 수 있습니다.
