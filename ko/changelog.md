# 변경사항 (What's New)

최신 버전이 맨 위에 있습니다. 앱은 켤 때마다 자동으로 최신 버전으로 업데이트됩니다 — 설치는 [다운로드](getting-started/download.md) 참고.

## v0.9.25

- 신규: 드롭 페이지 — 인기 체인에서 오픈 중이거나 갓 배포된 SeaDrop 민트를 스캔하고, 탭 한 번으로 스마트 민트에 바로 불러옵니다.
- 여러 지갑의 화이트리스트·서명 드롭 민팅이 빨라졌습니다: 지갑별 바우처를 병렬로, 드롭 단위로 묶어 받아 지갑이 많아도 하나씩 기다리지 않고 발사합니다.
- 신규: 한 드롭의 지갑별 작업이 작업 화면에서 접히는 카드 하나로 묶입니다 — 드롭·페이즈·지갑 수·실시간 진행을 보고 한 번에 전체 시작/정지.
- 신규: 작업별 가스 조정 — 개별 작업에 팁 배수와 최소 우선순위 수수료를 지정하고, 연사 모드용 15/30/60% 부스트 프리셋을 씁니다.
- 한국어·중국어 화면에서 영어로 보이던 일부 라벨을 바로잡았습니다.

## v0.9.24

- 신규: 화이트리스트(서명) 드롭에 내 OpenSea API 키를 쓸 수 있습니다. 한 번 등록하면 앱과 텔레그램 봇이 공용 SWAP 대신 OpenSea 공식 Drops API(계정당 분당 120회)로 바우처를 받습니다. 퍼블릭 민트는 여전히 키가 필요 없습니다.
- 신규: 프록시 테스트 URL을 직접 지정할 수 있습니다. 원하는 주소로 프록시를 점검하세요(기본 api.ipify.org).
- 신규: 지갑 화면에서 WETH를 ETH로 한 번에 되돌립니다(언랩).
- 신규: 앱·사용자 가이드·텔레그램 봇이 모두 중국어 간체(简体中文)를 지원합니다 — 설정에서 언어를 바꾸세요.
- 영어·한국어·중국어 전반의 문구를 더 자연스럽게 다듬었습니다.

## v0.9.23

- 신규: 지갑에 EIP-7702 위임(일부 민트가 거부하는 스마트계정 업그레이드)이 걸려 있는지 확인하고, 지갑 화면에서 한 번에 취소할 수 있습니다.
- 신규: 설정에서 로그아웃. 실행 중인 작업을 모두 멈추고 키로 앱을 잠그지만, 지갑·RPC·설정은 그대로 유지됩니다.
- 신규: 자금을 나눌 때 지갑마다 랜덤 금액으로 보낼 수 있어, 모든 지갑에 똑같은 금액이 찍히지 않습니다.
- 신규: 외부 주소(거래소 입금 주소나 콜드월렛)로 바로 취합할 수 있고, 가스 예상치를 ETH / USD로 한 번에 전환할 수 있습니다.
- 화이트리스트 민팅이 대규모에서 rate limit에 더 안전합니다: voucher 요청을 프록시에 나눠 보내고, 스팸 감시가 민트 오픈 시각을 더 촘촘히 추적합니다.
- 앱과 앱 내 가이드 전반의 문구를 더 자연스럽게 다듬었습니다.

## v0.9.22

- 스마트 민트에서 화이트리스트 / 서명 페이즈의 예상 비용이 정확히 표시됩니다 — 더 저렴한 명단 페이즈를 골라도 퍼블릭 가격이 뜨던 문제를 고쳤습니다. (화면 표시만; 실제 발사는 항상 올바른 가격이었습니다.)
- ERC-1155 NFT를 다른 지갑으로 보내기·나눠 보내기가 정상 동작합니다 — 이전엔 매번 되돌려지던 문제를 수정했습니다.
- 민팅·전송·리스팅·텔레그램 봇 전반의 안정성과 신뢰성을 폭넓게 개선했습니다.

## v0.9.21

- 화이트리스트 민팅이 훨씬 빨라졌습니다 — OpenSea가 바우처를 발급하는 즉시 포착하고 드롭이 열리는 순간 발사합니다 (이전엔 수 초 지연).
- 화이트리스트 확인이 이제 드롭의 실제 페이즈와 각 지갑의 명단 여부(✓ / ✗)를 표시합니다 (이전엔 '퍼블릭'만).
- 스마트 민트가 드롭의 올로우리스트 / 서명 페이즈를 정확히 읽어 표시합니다.
- 발사가 가스 안전합니다 — 솔드아웃되거나 아직 안 열린 드롭에 가스를 낭비하지 않습니다.
- 스팸 작업을 이더리움 메인넷에서 프라이빗·리버트 보호되는 Flashbots 번들로 발사할 수 있습니다 (선택).
- 드물게 발생하던 화면 멈춤을 수정하고 텍스트 대비 가독성을 개선했습니다.

## v0.9.20

- 부스트가 이제 거래를 더 쌓지 않고 제자리에서 교체합니다 — 같은 논스, 더 높은 가스. 실행 중인 작업에도 동작하며 +10% / +25% / +50% 빠른 프리셋을 제공합니다.
- 작업 만들 때 "자금 있는 지갑만" 빠른 선택 추가 — 민트할 잔액이 있는 지갑만 바로 고릅니다.
- 화이트리스트 스캔 결과를 저장·불러오기·삭제할 수 있습니다 — 다이얼로그를 닫아도 적격 결과가 사라지지 않습니다.
- 스팸 작업에 종료 시각을 설정할 수 있습니다 — 무제한으로 설정해도 지정한 시각에 자동으로 멈춥니다.
- 작업 에디터의 지갑 선택에 잔액이 표시되고, 시작 시각이 초 단위를 지원하며, 시작 / 중지 / 부스트 버튼을 정리했습니다.
- 스팸 타이밍이 더 안정적이고, 앱을 종료하면 실행 중인 모든 작업이 깔끔하게 정지됩니다.

## v0.9.19

- OpenSea 민트 버그 수정: 이미 시작된 드롭(토큰이 일부라도 민트된 경우)에서 민트 데이터를 못 받아 발사되지 않던 문제. 이제 퍼블릭·서명(GTD / FCFS 화이트리스트) 페이즈 모두 정상적으로 민트됩니다.

## v0.9.18

- 스팸 모드가 이제 가스 안전합니다. 실패할 거래는 더 이상 발사하지 않습니다 — 드롭이 아직 안 열렸거나·솔드아웃이거나·지갑이 이미 한도에 도달한 동안엔 가스 0으로 대기하며 이유를 알려주고, 민트가 실제로 될 수 있는 순간 바로 발사합니다.
- 발사할 때는 수백 개의 실패 거래로 체인을 도배해 솔드아웃 드롭에서 실제 가스가 나가던 방식 대신, 살아있는 거래 1개만 유지하고 안 들어가면 가스를 올려 교체합니다.
- 재시도 가스 부스트 프리셋(작업별), 오픈 순간 더 빠른 발사, 그리고 지연을 최소화하려면 안전 검사를 건너뛰고 바로 발사하는 옵션을 추가했습니다.

## v0.9.17

- 스팸 모드가 이제 화이트리스트 드롭에서 민트 데이터가 아직 준비 안 됐을 때 포기하지 않고, 페이즈가 열려 준비되는 순간 발사합니다. 어떤 페이즈(퍼블릭·GTD WL·FCFS WL)든 몇 분 미리 만들어두고 스팸으로 돌리면 열리는 순간을 잡습니다.

## v0.9.16

- 수정: 드롭이 열리기 전에 미리 만들어 둔 퍼블릭 민트가 실행 시 발사되지 않던 문제("function name or hex calldata required") — 이제 정상 발사되며, 스팸 모드는 드롭이 열릴 때까지 계속 재시도합니다.

## v0.9.15

- 화이트리스트 확인이 이제 본인 연결로 직접 실행됩니다 — 같은 드롭에 많은 사람이 몰릴 때 더 빠르고 안정적입니다.
- OpenSea 민트의 가스 여유를 늘렸습니다 (실제 사용한 가스만큼만 청구됩니다).
- 텔레그램 봇: 민트가 이제 본인 프록시로 나갑니다 — 봇 설정에서 프록시를 추가하세요(없으면 안내가 표시됩니다).

## v0.9.14

- 지갑·프록시·RPC·NFT 화면에 영어로 남아 있던 일부 문구를 한국어로 번역했습니다.
- 설정의 자동 가스 항목 이름을 알기 쉽게 정리하고, 발사 지연 입력에 권장값 안내를 추가했습니다.
- 진단용 로그 파일 추가 — 문제가 생기면 로그 폴더를 열어 보내주시면 지원에 도움이 됩니다.
- 민트 전 토큰(ERC-20) 승인이 더 안정적이고, 내부 안정성·보안 개선과 버그 수정을 적용했습니다.

## v0.9.13

- OpenSea 드롭이 OpenSea 사이트가 바뀌어도 끊김 없이 작동합니다 — 앱이 스스로 맞춰서 수동 업데이트가 필요 없습니다.
- 여러 지갑으로 하는 OpenSea 민팅이 더 빠르고 안정적입니다.
- 토큰(USDC 등)으로 결제하는 드롭: 민트 전에 토큰을 승인하는 버튼을 추가했습니다.
- NFT 리스팅을 개별로 취소할 수 있습니다 — 지갑 전체를 한꺼번에 취소하는 것 외에 선택만 취소.
- 예약 민트가 정확한 시각에 더 안정적으로 발사되고, 여러 안정성·안전성 개선과 버그 수정.

## v0.9.12

- 대시보드를 2열로 정리 — 위: 드롭·감시 목록 / 아래: 민트 피드·실행 중.
- 이제 업데이트 소식을 한국어·영어로 텔레그램에서도 받습니다.

## v0.9.11

- 스마트 민트 새 화면 — 링크만 붙여넣으면 드롭·가격·페이즈·지갑별 적격을 한눈에.
- 드롭 이미지, 오픈 카운트다운, 솔드아웃·사칭(가짜 이름) 경고 추가.
- 민트할 지갑·페이즈·가스를 직접 고르고, 예상 총비용까지 표시.
- 프록시 그룹: 발사와 voucher 요청이 같은 프록시 IP로 나갑니다.
- 민팅 후 안전 지갑으로 자동 스윕, 최근·감시 드롭 목록 추가.
- 발사를 막지 않습니다 — 적격이 아니거나 잔액이 부족한 지갑도 그대로 발사하고, 성공·실패는 네트워크가 판단합니다.
