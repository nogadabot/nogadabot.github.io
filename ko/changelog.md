# 변경사항 (What's New)

최신 버전이 맨 위에 있습니다. 앱은 켤 때마다 자동으로 최신 버전으로 업데이트됩니다 — 설치는 [다운로드](getting-started/download.md) 참고.

## v0.9.19

- OpenSea 민트 버그 수정: 이미 시작된 드롭(토큰이 일부라도 민트된 경우)에서 민트 데이터를 못 받아 발사되지 않던 문제. 이제 퍼블릭·서명(GTD / FCFS 화이트리스트) 페이즈 모두 정상적으로 민트됩니다.

## v0.9.18

- 스팸 모드가 이제 가스 안전합니다. 실패할 거래는 더 이상 발사하지 않습니다 — 드롭이 아직 안 열렸거나·솔드아웃이거나·지갑이 이미 한도에 도달한 동안엔 가스 0으로 대기하며 이유를 알려주고, 민트가 실제로 될 수 있는 순간 바로 발사합니다.
- 발사할 때는 수백 개의 실패 거래로 체인을 도배해 솔드아웃 드롭에서 실제 가스가 나가던 방식 대신, 살아있는 거래 1개만 유지하고 안 들어가면 가스를 올려 교체합니다.
- 재시도 가스 부스트 프리셋(작업별), 오픈 순간 더 빠른 발사, 그리고 지연을 최소화하려면 안전 검사를 건너뛰고 바로 발사하는 옵션을 추가했습니다.

## v0.9.17

- 스팸 모드가 이제 화이트리스트 드롭에서 민트 데이터가 아직 준비 안 됐을 때 포기하지 않고, 페이즈가 열려 준비되는 순간 발사합니다. 어떤 페이즈(퍼블릭·GTD WL·FCFS WL)든 몇 분 미리 만들어두고 스팸으로 돌리면 열리는 순간을 잡습니다.

## v0.9.16

- 수정: 드롭이 열리기 전에 미리 만들어 둔 퍼블릭 민트가 실행 시 발사되지 않던 문제("function name or hex calldata required") — 이제 정상 발사되며, 스팸 모드는 드롭이 열릴 때까지 계속 재시도합니다.

## v0.9.15

- 화이트리스트 확인이 이제 본인 연결로 직접 실행됩니다 — 같은 드롭에 많은 사람이 몰릴 때 더 빠르고 안정적입니다.
- OpenSea 민트의 가스 여유를 늘렸습니다 (실제 사용한 가스만큼만 청구됩니다).
- 텔레그램 봇: 민트가 이제 본인 프록시로 나갑니다 — 봇 설정에서 프록시를 추가하세요(없으면 안내가 표시됩니다).

## v0.9.14

- 지갑·프록시·RPC·NFT 화면에 영어로 남아 있던 일부 문구를 한국어로 번역했습니다.
- 설정의 자동 가스 항목 이름을 알기 쉽게 정리하고, 발사 지연 입력에 권장값 안내를 추가했습니다.
- 진단용 로그 파일 추가 — 문제가 생기면 로그 폴더를 열어 보내주시면 지원에 도움이 됩니다.
- 민트 전 토큰(ERC-20) 승인이 더 안정적이고, 내부 안정성·보안 개선과 버그 수정을 적용했습니다.

## v0.9.13

- OpenSea 드롭이 OpenSea 사이트가 바뀌어도 끊김 없이 작동합니다 — 앱이 스스로 맞춰서 수동 업데이트가 필요 없습니다.
- 여러 지갑으로 하는 OpenSea 민팅이 더 빠르고 안정적입니다.
- 토큰(USDC 등)으로 결제하는 드롭: 민트 전에 토큰을 승인하는 버튼을 추가했습니다.
- NFT 리스팅을 개별로 취소할 수 있습니다 — 지갑 전체를 한꺼번에 취소하는 것 외에 선택만 취소.
- 예약 민트가 정확한 시각에 더 안정적으로 발사되고, 여러 안정성·안전성 개선과 버그 수정.

## v0.9.12

- 대시보드를 2열로 정리 — 위: 드롭·감시 목록 / 아래: 민트 피드·실행 중.
- 이제 업데이트 소식을 한국어·영어로 텔레그램에서도 받습니다.

## v0.9.11

- 스마트 민트 새 화면 — 링크만 붙여넣으면 드롭·가격·페이즈·지갑별 적격을 한눈에.
- 드롭 이미지, 오픈 카운트다운, 솔드아웃·사칭(가짜 이름) 경고 추가.
- 민트할 지갑·페이즈·가스를 직접 고르고, 예상 총비용까지 표시.
- 프록시 그룹: 발사와 voucher 요청이 같은 프록시 IP로 나갑니다.
- 민팅 후 안전 지갑으로 자동 스윕, 최근·감시 드롭 목록 추가.
- 발사를 막지 않습니다 — 적격이 아니거나 잔액이 부족한 지갑도 그대로 발사하고, 성공·실패는 네트워크가 판단합니다.
