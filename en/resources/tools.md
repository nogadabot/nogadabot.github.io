# Gas Trackers & Useful Tools

External tools worth bookmarking when minting.

## ⛽ Check gas prices

* **Etherscan Gas Tracker**: [etherscan.io/gastracker](https://etherscan.io/gastracker), live Ethereum gas.
* **Blocknative Gas Estimator**: [blocknative.com](https://www.blocknative.com), gas prediction.

> 💡 Also use Nogada's own [gas calculator](../app-guide/tools.md) and the live gas in the bottom status bar.

## 🔍 Reading contracts

* **Etherscan**: [etherscan.io](https://etherscan.io), view a contract's mint function (Write Contract) and other transactions' input data (hex). Essential for inspecting functions and decoding hex input data.

> 💡 Confused about a function/hex? Ask **ChatGPT / Claude** "what's this contract's mint function and arguments?", modern AI reads contracts well.

## 📊 Wallet/NFT tracking

* **NFTBank**: [nftbank.ai](https://nftbank.ai), track a wallet's NFT portfolio & value.

> 💡 You can also check inside Nogada via [NFT Holdings](../app-guide/nft.md) and [PnL](../app-guide/pnl.md).

## 🌐 Free RPC

* **Chainlist**: [chainlist.org](https://chainlist.org), public RPC addresses per chain.

> ⚠️ External links change often. If a link is down, search by name.
